This is a very smple utility to help during validation or qualification process.

Prepare an Excel file containing test code and description as described below

Code        | Description
1_test_code | 1_test desctiption
2_test_code | 2_test desctiption
...         | ...

Open the file with the app and you will have the possibility to choose a test and 
capture the screen.
Automatically the file will be saved in the same directory of the Excel file using 
the test code as name, followed for a sequencial number is the name is present.

The application will show a preview of the last saved image, to see directly if
the capture was satisfatory.
